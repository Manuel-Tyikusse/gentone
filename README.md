# Gentone - AI-Powered SaaS Platform

Gentone é uma plataforma SaaS (Software as a Service) moderna desenvolvida para demonstrar uma arquitetura escalável utilizando renderização no lado do servidor (SSR) e gerenciamento seguro de middleware.

## 🛠️ Tecnologias Utilizadas
- **Framework:** Next.js (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Segurança:** Next.js Middleware para controle de sessões e rotas protegidas

## 🚀 Funcionalidades Planejadas / Implementadas
- Autenticação de usuários segura.
- Arquitetura de componentes modulares e reutilizáveis na pasta `/components`.
- Isolamento de lógica de negócios e chamadas de API na pasta `/lib`.

## ⚙️ Como Executar o Projeto
1. Clone o repositório: `git clone https://github.com/Manuel-Tyikusse/gentone.git`
2. Instale as dependências: `npm install`
3. Execute o servidor de desenvolvimento: `npm run dev`
